# Business objective

Provide an API endpoint by which a prediction of optimal performance for a given form, providing the expected results and recommended configuration to achieve it.

# Requirements

1. Provide hosted model endpoint via API with failure recovery/mitigation
2. Model ought to be automatically updated on a regular schedule (weekly?)
3. Logging of API requests to include inputs, time to response, value of response
4. Logging of model re-trainings to monitor test accuracy over time
