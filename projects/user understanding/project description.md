# User adoption/understanding metric

## Business objective

Quantify (1) feature adoption and (2) feature understanding by clients 

## Completion

- Define a point of feature and platform understanding
- Develop understanding metric deliverable to CX
- Discover client difficulties in feature adoption, possibly highlighting features clients are more likely to struggle to deploy
- Report potentially difficult feature list to CX
- Develop adoption metric deliverable to CX

## Milestones

- Identify likely data sources (1 week)
- Exploring and transforming the data (3 weeks)
- Develop metrics and recommendations (1 week)
- Develop understanding & adoption metrics in independent form (functions to calculate value in production)
